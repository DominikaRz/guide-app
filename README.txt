Mobile City Guide

1.The app should be able to use GPS or similar technologies to pinpoint the user's current location and provide relevant information accordingly.

2.App should include restaurants, parks, museums, theaters, landmarks, shopping centers, hospitals, police stations, etc. (min. 10 places)

3.Each location could have additional information like opening hours, contact info, user ratings, and reviews.

4.Based on places create some city tour proposals for 4h, one day, two days of visit.

5.App should provide interactive map which shows all mentioned places
